# Language Learning App 🌍

A dynamic web-based language learning application that helps users learn multiple languages through interactive flashcards and quizzes.

## 🌟 Features

### Multiple Languages
- Spanish 🇪🇸
- French 🇫🇷
- German 🇩🇪
- Italian 🇮🇹

### Interactive Learning Tools
- 📝 Flashcards with word-translation pairs
- ✍️ Interactive quizzes
- 🔊 Text-to-speech pronunciation
- 📊 Progress tracking
- ➕ Custom word addition feature

### User Experience
- 🎨 Modern, clean interface
- 📱 Responsive design
- ⏱️ Timed quizzes
- 💾 Progress saving

## 🚀 Getting Started

1. Visit the app at: yourusername.github.io/language-learning-app
2. Select your preferred language
3. Choose between Flashcard or Quiz mode
4. Start learning!

## 📖 How to Use

### Flashcard Mode
1. Select your language
2. Click 'Flip' to see translations
3. Use 'Next' for new words
4. Click 🔊 for pronunciation

### Quiz Mode
1. Choose your language
2. Select the correct translation
3. Submit within time limit
4. Track your score
